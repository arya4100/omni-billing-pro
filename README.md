# OmniBilling Pro

OmniBilling Pro is an all-in-one business billing, invoicing, and transaction management suite for Odoo 16. It offers a classic and highly functional design intended as a premium product on the Odoo Apps Store.

## Features

* **Advanced Invoice Management:** Create and manage invoices efficiently.
* **Unified Transaction Dashboard:** Track granular payments related to specific invoices.
* **Partial Payments:** Easily manage remaining balances automatically.
* **Clean & Professional UI:** Built to give users a premium experience.
* **Custom Sequences:** Automated clean reference numbers for invoices (INV/YYYY/0000X) and transactions (TRX/YYYY/0000X).

## How to Install

1. Move the `omni_billing_pro` directory to your Odoo `addons` path.
2. Restart your Odoo server.
3. Enable **Developer Mode** (Settings -> Activate developer mode).
4. Go to **Apps** -> **Update Apps List**.
5. Search for `OmniBilling Pro` and click **Install**.

## Business Use Case

Any business from small to enterprise looking to move away from rigid standard accounting to a more streamlined and flexible Billing/Accounts Receivable model will benefit greatly from this module.
